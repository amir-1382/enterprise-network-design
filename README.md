# enterprise-network-design
My project compares three interior gateway routing protocols – RIP, OSPF, and EIGRP – under realistic campus WAN conditions. I designed a 9‑router topology representing the different sites, with redundant links of varying bandwidths (1 Gbps, 100 Mbps, 10 Mbps). I implemented the network in Cisco Packet Tracer (for initial validation) to overcome serial interface limitations. The simulation includes switches, a central DHCP server (dedicated server in Packet Tracer), and basic security (passwords, SSH). Using Wireshark, I measure throughput, end‑to‑end delay, convergence time (link and router failures), routing overhead, path selection, and load balancing for each protocol. The goal is to provide empirical evidence on which protocol performs best in different network scenarios, helping network engineers make informed choices.
